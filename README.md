# Battleship

**Play the live game here:** [roaayala.github.io/battleship](https://roaayala.github.io/battleship/)

A classic Battleship game built for the web. This project focuses on applying modern frontend architecture, robust state management, and test-driven development without relying on UI frameworks.

---

## Technology and Approach

- **Architecture:** Model-View-Controller (MVC)
- **Build Tools:** Webpack and Babel
- **Testing:** Unit testing with Jest
- **JavaScript:** Vanilla JS featuring Component-Based UI and Drag-and-Drop API
- **Styling:** CSS Grid, Flexbox, and CSS Variables with a Mobile-First approach
